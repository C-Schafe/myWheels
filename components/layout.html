# Layout 默认布局

通过不同容器的组合快速实现页面区域的划分。

# 常见布局

基本容器为：<w-layout>、<w-header>、<w-content>、<w-footer>、<w-sider> 。通过这几个基本的布局容器的组合，可以实现常见的页面布局形式。
