# Collapse 折叠面板

可以折叠 / 展开的内容区域。

# 基础用法

通过 w-collapse 和 w-collapse-item 引入折叠面板，w-collapse 的 selected 属性配置默认选中项，w-collapse-item 的 name 和 title 为必填项，确定面板的名称和标题。

# 单项展开

w-collapse 上的 single 属性可配置展开面板仅为一项。
